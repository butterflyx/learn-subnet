# learn-subnet

bash script to learn subnetting with random IPs


## Install

### clone repository
```
$ git clone https://github.com/butterflyx/learn-subnet.git
```

### make it executable
```
$ cd learn-subnet/
~/learn-subnet$ chmod +x learnip.sh
```

### use it
![learn-subnet](https://user-images.githubusercontent.com/5388631/87831400-69bb2500-c851-11ea-85ac-46425b46f5f1.png)


## offline results

this package includes the perl-script "ipcalc" from Krischan Jodies in the lib folder.
Use the `-t [time]` parameter to set time until solutions will be displayed.

You can use the time values as for the Unix `sleep` command:
- s for seconds (can be omited)
- m for minutes.
- h for hours.
- d for days.
