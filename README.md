# learn-subnet

simple bash script to learn subnetting with random IPs


## Install

### clone repository
```
$ git clone https://github.com/butterflyx/learn-subnet.git
```

### make it executable
```
$ cd learn-subnet/
~/learn-subnet$ chmod +x learnip.sh
```
